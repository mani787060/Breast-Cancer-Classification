# Breast Cancer Classification using Logistic Regression  
Binary Classification | Machine Learning Project

This project implements a **Breast Cancer Classification system** using the  
**Logistic Regression** algorithm to classify tumors as **Malignant** or **Benign**  
based on diagnostic features.

---

## Project Overview
- Load and explore the Breast Cancer dataset  
- Perform data preprocessing and feature scaling  
- Split data into training and testing sets  
- Train a Logistic Regression classifier  
- Evaluate model performance using accuracy  
- Build a predictive system for new patient data  

---

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

---

## Algorithm Used
### Logistic Regression
- Suitable for binary classification  
- Interpretable and efficient  
- Widely used in medical diagnosis  

---

## Dataset Information
- Dataset: Breast Cancer Wisconsin Dataset  
- Target Classes:
  - 0 → Malignant  
  - 1 → Benign
